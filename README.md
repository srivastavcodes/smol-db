# Smol-Db
> This is a placeholder document, I'll add a real one when the program is useable. 

Smol-Db is a database built for someone who is looking to learn how to write a simple but functional relational database from
scratch using as little dependencies as possible; with extensive documentation of the code and how it works overall.

## Features
- core features of realtional databases such as `Create`, `Insert`, `Select`, etc.
